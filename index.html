Subway Quetz

Description:

Subway Quetz is a game of dodging obstacles whose main character is Quetzalcoatl himself. Your scenery will be the desert.


Developer:SAHARA team


Engine:3D Unity


Genre:Endless runner

OBSTACLE SPAWNER

It works by passing in the reference to the existing game object which is then duplicated as a clone.

MOVE PLAYER

The Transform component determines the position of the object in the scene via input keys.

TRIGGER / TIMER

Collider is setup as a Trigger so that it will use the event system / Storing a float time value to count up.

Street Danger

Description:

Street Danger is a game where the protagonist is forced to fight to save his city. Soldiers imperials will be your enemies.


Developer:Ositos Cariñositos team


Engine:2D Unity


Genre:Beat 'em up

LIFE SYSTEM

Event function removes a life each time the player is hit and ends the game when lives run out.

STATE MACHINE

It allows user input to trigger different animations and provides a way to overview all of the animation clips.

ENEMY TARGET

The vectors will determine how the enemy should move and the range at which the player is a target.
